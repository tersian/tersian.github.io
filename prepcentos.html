1.  Update your system with: sudo yum -y update

sudo setenforce 0

sudo sed -e s/SELINUX=enforcing/SELINUX=permissive/g /etc/selinux/config > /tmp/config.tmp;/usr/bin/cp -f /tmp/config.tmp /etc/selinux/config

2.     Disable selinux enforcement for now:

3.     Edit /etc/sysconfig/network-scripts/ifcfg-* files to change onboot=no to onboot=yes for your interfaces

4.     Run the command systemctl restart network

5.  Check network connectivity by running: ping 8.8.8.8

6.     Check your interfaces to see if theyve both received ip addresses (if you are on a network that serves DHCP).

# hostnamectl set-hostname openstack.training.lab

8.     Add your IP and hostname to /etc/hosts:

For example:

9.     Check /etc/resolv.conf for the nameserver directive, if it is not there, add one with your nameserver (if you have one, otherwise use 8.8.8.8 or 1.1.1.1)
